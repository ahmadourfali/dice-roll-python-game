# dice-roll-python-game
simple dice roll game using python
This is a very simple dice roll game using python. user is asked at the begining if they want to play or quit. If they play, they are asked for the number of dices to roll. After chosing a number of dices, random numbers between 1-6 are generated and displayed accoring to the chosen number of dices. If user quits, the game displays how many times they played.
